# Pi, Obsidian, & Tessl - an ultra powerful agent and coding framework

Pi is much more than just the agent framework that powers OpenClaw:
it's a powerful, plugable, and extensible harness that you can use for coding,
building bespoke agent workflows, running automations, and much more.
Combined with Obsidian and Tessl for context and graph knowledge,
you can build a powerful customized agent that works exactly how
you want with the context you need to accomplish large tasks.

In this talk, we'll go deep on using Pi, building and enabling extensions,
customization strategies, hooking it up with API based inference providers
as well as local models, using it with orchestration tools, and integrating
Obsidian with the Obsidian CLI and Tessl to enable your agents to
efficiently traverse a versioned graph of knowledge and handle agent
context across many distributed Pi agents with access to an Obsidian vault.
